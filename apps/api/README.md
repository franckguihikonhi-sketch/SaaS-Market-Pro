# Market-Pro API

Backend NestJS + Prisma + PostgreSQL (Supabase) du SaaS Market-Pro. Voir le
[README racine](../../README.md) pour le démarrage complet et
[docs/ARCHITECTURE.md](../../docs/ARCHITECTURE.md) pour l'organisation du code.

```bash
pnpm --filter api start:dev   # http://localhost:3001, docs sur /docs
pnpm --filter api test
pnpm --filter api test:e2e
pnpm --filter api exec prisma studio
```
